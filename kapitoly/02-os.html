Kapitola 2: Jak funguje počítač a operační systém

Hardware a software bez magie

Hardware

CPU, RAM a disk jsou základní stavební kameny počítače.

Software

Programy, které na hardware běží.

Operační systém

Operační systém řídí hardware a poskytuje prostředí pro aplikace.

Kernel

Kernel je jádro operačního systému a má plnou kontrolu nad hardwarem.
